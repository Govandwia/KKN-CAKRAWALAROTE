# Rote NTT — The Hidden Paradise of Eastern Indonesia

Welcome to the **Rote NTT** project! This is a modern, responsive, and visually stunning web application dedicated to showcasing the beauty, culture, tourism, and local products (UMKM) of Rote Island, East Nusa Tenggara, Indonesia.

## 🌟 Features

- **Beautiful Hero Section**: Engaging landing view with animated gradients and key statistics.
- **Interactive Map**: Discover hidden gems, beaches, surf spots, and cultural landmarks using an interactive Leaflet map with custom markers.
- **Dynamic Weather & Season Guide**: Live local time and seasonal recommendations for the best time to visit.
- **Tourism & Destinations Gallery**: Highlight top destinations in Rote with interactive cards and categories.
- **Culture & Heritage Showcase**: Information about the Sasando harp, Tenun Ikat, and local dances.
- **UMKM Catalog**: Connect with local makers and explore traditional crafts, culinary products, and accessories.
- **Event Calendar**: Stay updated with upcoming festivals and community events.
- **Fully Responsive**: Optimized for both desktop and mobile viewing.

## 🚀 Tech Stack

- **React 18**: Frontend UI library
- **TypeScript**: Static typing for robust code
- **Vite**: Next-generation frontend tooling for blazing-fast builds
- **Leaflet & React-Leaflet**: Open-source interactive maps
- **Vanilla CSS**: Custom styling with deep green/dark mode aesthetics, micro-animations, and glassmorphism.

## 🛠️ How to Run Locally

To get this project up and running on your local machine, follow these steps:

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <your-repo-url>
   cd rote
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to the local URL provided by Vite (usually `http://localhost:5173/`).

## 📁 Project Structure

- `src/components/`: Contains reusable React components (Hero, Map, Weather, Gallery, etc.)
- `src/pages/`: Contains the main page views (HomePage, ArticlePage)
- `src/data/`: Contains `site-content.json` which acts as the local database/CMS for the text, coordinates, and images used across the site.
- `src/index.css`: The central stylesheet containing all the UI styling and animations.
